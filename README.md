## Pré-requis
- Maven
- Docker
- Npm, Node

## Setup
- `chmod +x setup.sh stop.sh`
- `./setup.sh` Pour lancer les serveurs, le RP, le Message broker
- Se rendre sur `localhost:5173` sur son navigateur
- `./stop.sh` pour tout arrêter
